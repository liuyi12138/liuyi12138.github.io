markdown 基本用法

1.标题

在 Markdown 中，如果一段文字被定义为标题，只要在这段文字前加 # 号即可。
有多少个#即为几级标题。

TIPS : 在Git中 ‘#’ 后需加上空格

2.列表

在 Markdown 中，列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，
有序列表则直接在文字前加1. 2. 3. 符号要和文字之间加上一个字符的空格。

  • 1.
  • 2.
  • 3

Tips ： ‘-‘和’*‘在同一个无序列表可以混用
TIPS ： 空一格风味更佳哦

    • 2.
    • 3.

在Markdown中，在一段文字前加上’>’即将这一段话定义为引用。
